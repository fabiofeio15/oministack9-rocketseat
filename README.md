# Aplicação (aircnc) da Semana Oministack 9 da Rocketseat

Código desenvolvido prática deliberada de programação durante a semana Omnistack 9 da Rocketseat.

Sistema permite 'empresas' cadastrarem 'spots' para reservas e receber notificações em tempo real entre a aplicação web e mobile.

</p>

## Recursos Utilizados

- **[Node](https://nodejs.org/en/)**
- **[Mongo](https://www.mongodb.com/cloud/atlas)**
- **[Express para API](https://expressjs.com/)**
- **[React](https://reactjs.org/)** para aplicação front-end web e mobile
- **[Expo](https://expo.io/)** para build aplicação React Native
- **[Socket.io](https://socket.io/)**
